# DOCUMENT ORGANISATION

### **début du dev de subrunner : 19/11/2023**


Ce document a pour but d'aider le développement du jeu [**subrunner**](https://github.com/louisaiva/subrunner)

Ce document est séparé en plusieurs sous parties, chacune ayant un role défini. On retrouve notamment :

# [[GDD]]
- (**game design document**)
- situé dans `/high_level (gdd)`
- design high level du jeu

# [[TD]]
- (**technical document**)
- situé dans `/low_level (td)`
- design low level du jeu, peut servir de documentation aussi pour les futur.e.s developpeureuses

# Journaux
- situé dans `/journals`
- garde une trace des journaux écrits au jour le jour

# Resources
- situé dans `/resources`
- stocke toutes les images/fichiers utiles/excalidraw etc

# Templates
- situé dans `/templates`
- stocke tous les templates obsidian utiles au document
# Logs
- situé dans `/logs`
-  (moins organisé lol) contient les différents /logs et autres fichiers vieux, deprecated etc
